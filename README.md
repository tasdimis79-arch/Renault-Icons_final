RENAULT TBOX ICON PACK - READY FOR GITHUB BUILD

How to build in GitHub:
1. Extract this zip.
2. Upload everything to a new GitHub repository.
3. Open Actions.
4. Run workflow: Build Android APK
5. Download the artifact named: renault-icon-pack

Notes:
- This project includes these icons:
  YouTube, Spotify, Maps, Phone, Radio, Torque, Car, Apps, File Manager,
  Settings, Play Store, Chrome, Music, Gmail
- Automatic app mapping should work for common Android apps.
- For custom head-unit apps like Radio / Car / Apps, you may still need to
  assign the icon manually inside your launcher if the package name differs.
